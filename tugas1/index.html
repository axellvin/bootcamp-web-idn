About Me

Halo, perkenalkan nama saya Axel Leovincent. Saya berkuliah di Universitas Multi Data Palembang. Saya memiliki minat yang besar untuk dapat berkarir di dunia IT, khususnya sebagai Web Developer. Selama berkuliah saya mengamati dengan baik, bahwa seorang Web Developer membutuhkan skill dalam membuat tampilan Web yang menarik dan responsive yang disebut Frontend Web sedangkan untuk memproses request, mengelola dan menyimpan data. Saya sudah mempelajari dengan baik mengenai Front End Web dan saya sudah menguasai HTML, CSS, Java Script, dan Framework CSS (Bootstrap & Tailwind), JS (AngularJS). Saya juga sudah mempelajari dan menguasai Back End Web dan bahasa pemrograman/Framework seperti, PHP (Code Igniter & Laravel), JS(Node.js), dan Java (Spring), serta database yang saya kuasai yaitu MySQL, Microsoft SQL Server dan MongoDB.
